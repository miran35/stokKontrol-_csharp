
Stok Kontrol Projesi

Bu proje, temel C# programlama konularını pekiştirmek amacıyla geliştirilmiş basit bir stok kontrol uygulamasıdır.
Projede class yapıları, if-else koşulları ve switch-case kullanılmıştır.

Özellikler
Güncel stok görüntüleme
Stoğa ürün ekleme
Stoktan ürün satma/düşürme
Çıkış işlemi

Bu proje sayesinde nesne yönelimli programlama mantığı ve temel kontrol yapıları uygulamalı olarak uygulanmıştır.
